# Roadmap

## Сейчас · 1.0.0

- локальный контекстный граф для кода и документов;
- официальный онлайн-установщик с проверкой подписи релиза;
- поддержка распространённых AI-инструментов;
- Trial, коммерческие и исследовательские лицензии.

## Ближайшие направления

- публичные примеры интеграции и наборы для воспроизводимых проверок;
- каталог совместимых платформ и форматов;
- документация для внешних адаптеров;
- публичная политика доступности исходного кода.

## Как влиять на roadmap

Откройте Issue с задачей, материалом, который нужно подключить, и AI-инструментом. Приоритет получают повторяемые сценарии с понятным эффектом: меньше повторного чтения, быстрее поиск основания вывода, точнее проверка изменений.
