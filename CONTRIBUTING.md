# Contributing

До открытия исходного кода полезны:

- отчёты об ошибках с минимальным воспроизводимым примером;
- запросы на платформы, форматы документов и адаптеры;
- замечания к документации и сценарии использования;
- предложения по benchmark-наборам.

Не прикладывайте документы клиентов, лицензии, токены, персональные данные, готовые коммерческие дистрибутивы или закрытый код Sphaera.

Для каждого сообщения укажите версию Sphaera, ОС, подключённый AI-инструмент и ожидаемый результат.
