# Security policy

Не публикуйте уязвимости, ключи, токены установки, данные клиентов или внутренние логи в Issues.

Отправьте краткое описание, шаги воспроизведения и возможное влияние на `info@iustitia.tech` с темой `Sphaera security report`. Мы подтвердим получение и сообщим дальнейший порядок работы.
