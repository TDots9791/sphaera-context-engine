# Contributing

Before the source code is opened, useful contributions include:

- bug reports with a minimal reproducible example;
- requests for platforms, document formats, and adapters;
- documentation feedback and real-world use cases;
- proposals for benchmark suites.

Do not attach customer documents, licenses, installation tokens, personal data, commercial distributions, or private Sphaera code.

For each report, include the Sphaera version, operating system, connected AI tool, expected result, and actual result.
