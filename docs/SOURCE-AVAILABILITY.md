# Доступность исходного кода

Коммерческий исходный код Sphaera context engine пока не опубликован. Его текущая лицензия запрещает публичное размещение и производные продукты.

Этот репозиторий создаёт публичный контур: документация, примеры, вопросы совместимости, roadmap и обратная связь. Перед публикацией кода будет отдельно определена модель лицензирования, чтобы не ослабить защиту пользователей, подписанных релизов и коммерческой поддержки.

До этого момента единственный официальный канал установки — сайт Iustitia.
