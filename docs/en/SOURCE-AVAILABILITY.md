# Source availability

The commercial source code of Sphaera context engine is not public yet. Its current license prohibits public placement and derivative products.

This repository provides a public contour: documentation, examples, compatibility questions, roadmap, and feedback. Before source code is published, the licensing model will be defined separately so that it does not weaken protection for users, signed releases, or commercial support.

Until then, the Iustitia website is the only official installation channel.
