# Official installation

Sphaera is installed only through the official online installer.

1. Open https://iustitia.tech/academy/sphaera-context-engine/.
2. Activate a Trial or purchase a license.
3. In your Sphaera account, copy the personal command for Windows or Linux.
4. Run it in PowerShell or a terminal.

The command is time-limited and links installation to your license and device. It verifies the signature of the downloaded release, installs Sphaera, and connects selected AI tools.

Do not download distributions from unofficial sources or publish your personal installation command: it contains a temporary issuance token.
