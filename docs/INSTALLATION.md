# Официальная установка

Sphaera устанавливается только через официальный онлайн-установщик.

1. Откройте https://iustitia.tech/academy/sphaera-context-engine/.
2. Активируйте Trial или оформите лицензию.
3. В личном кабинете Sphaera скопируйте персональную команду для Windows либо Linux.
4. Запустите её в PowerShell или терминале.

Команда действует ограниченное время и связывает установку с вашей лицензией и устройством. Она проверяет подпись скачанного релиза, устанавливает Sphaera и подключает выбранные AI-инструменты.

Не скачивайте дистрибутивы из неофициальных источников и не публикуйте персональную команду установки: в ней содержится временный токен выдачи.
